### 一、信息收集 (Information Gathering)

#### 1.1 子域名枚举与发现 (Subdomain Enumeration & Discovery)

*   **字典枚举/爆破:**
    *   **dnsenum**: 使用字典枚举子域名。(`dnsenum --enum <域名>`)
    *   **Layer 子域名挖掘机**: 图形化子域名爆破工具。
    *   **SubDomainBrute**: 高性能子域名爆破工具。
    *   **OneForAll**: 功能强大的子域名收集工具（也利用其他方法）。
*   **聚合搜索引擎/公开数据:**
    *   **Sublist3r**: 利用搜索引擎和公开数据源（VirusTotal, DNSDumpster, Netcraft 等）枚举子域名。
    *   **OneForAll**: (同上)
    *   **Amass**: (OWASP) 功能全面的子域名枚举和网络映射工具（也利用其他方法）。
    *   **Findomain**: 快速的子域名枚举工具，常用于 CT 日志查询。
*   **被动/历史数据:**
    *   **DnsDB / 历史 DNS 查询**: 查询历史 DNS 解析记录。
        *   *在线平台*: ViewDNS.info, Threatbook, SecurityTrails, `https://dnsdb.io/`
    *   **证书透明度 (Certificate Transparency - CT Logs)**: 查询 CT 日志发现已签发证书的子域名。
        *   *在线平台*: `https://crt.sh/`, `https://censys.io/certificates`
        *   *工具*: **Findomain**, **Amass**
*   **从源码/JS 文件中提取:**
    *   **JSFinder**: 扫描 JS 文件提取子域名、URL、IP。
    *   **LinkFinder**: 分析 JS 文件挖掘端点、参数和链接。
    *   **URLFinder**:  同上。

#### 1.2 C 段与旁站扫描 (C-Segment & Adjacent Site Scanning)

*   **K8_C段旁注查询工具**: 扫描指定 IP 的 C 段，收集旁站信息。
*   **IISPutScanner**: 扫描 C 段检查 IIS PUT 漏洞或识别信息。

#### 1.3 WHOIS 与备案查询 (WHOIS & ICP Lookup)

*   **WHOIS 查询**: 获取域名注册信息。
    *   *命令行*: `whois` (Kali Linux)
    *   *在线平台*: 站长之家, 爱站, ip138, ICANN Lookup, 腾讯云/阿里云 WHOIS, 新网 WHOIS。
*   **备案号查询**: 获取网站备案主体信息。
    *   *在线平台*: ICP备案查询网 (`icp.bugscaner.com`), 站长工具 ICP, 爱站 ICP, 工信部备案管理系统 (`beian.miit.gov.cn`)。

#### 1.4 敏感信息与泄露探测 (Sensitive Information & Leakage Detection)

*   **目录与文件扫描:**
    *   **DirSearch**: 基于 Python 的目录/文件扫描器。
    *   **Gobuster**: Go 语言编写，支持目录/文件、DNS、VHost 扫描。
    *   **ffuf**: Go 语言编写，快速 Web Fuzzing 工具，可用于目录/文件扫描。
    *   **dirbuster**: OWASP 出品，Java 图形化目录扫描工具。
    *   **wfuzz**: 强大的 Web Fuzzing 工具。
    *   **DirMap**: Go 语言编写的 Web 目录扫描器。
    *   **7kbscan (WebPathBrute)**: 国产 Web 路径扫描工具。
    *   **御剑后台扫描**: 经典的图形化后台/目录扫描工具。
    *   **Test404轻量后台扫描器**: 国产轻量级后台扫描工具。
    *   **Web破壳工具**: (名称较泛，可能指特定扫描器或集成工具)。
    *   **(间接使用) Nikto**: 漏洞扫描器，但也包含目录/文件检查功能。
    *   **(间接使用) WhatWeb**: 指纹识别工具，`-a 3` (aggressiveness) 模式可能包含路径探测。
*   **备份/源码泄露扫描与恢复:**
    *   **备份文件扫描器**: **Test404网站备份文件扫描器**, **ihoneyBakFileScan**, **BBScan** (综合性扫描，包含备份文件检测)。
    *   **.git 泄露利用**: **GitHack**, **dvcs-ripper**, **SourceLeakHacker**.
    *   **.svn 泄露利用**: **SVN Gopher**, **Seay SVN Exploiter**, **dvcs-ripper**.
    *   **.DS_Store 泄露利用**: **ds_store_exp**.
*   **代码托管平台搜索 (GitHub等):**
    *   **目的**: 查找泄露的源码、密钥、密码、配置。
    *   **工具**: **GSIL (Git Source Information Leak)**, **Github-Monitor**, **truffleHog**, **GitGuardian** (商业服务)。
*   **邮箱地址收集:**
    *   **The Harvester**: 聚合多种公开资源收集邮箱、子域名等。
    *   **Infoga**: 专注于从多种来源收集邮箱信息。
    *   *在线平台/方法*: **Hunter.io**, WHOIS, 网页爬取, GitHub, 搜索引擎, 数据泄露库 (`haveibeenpwned.com`, `monitor.mozilla.org`)。
*   **公网网盘/文档搜索:**
    *   *平台示例*: 凌云风搜索, 大力盘, 小不点搜索等。
*   **历史网页快照:**
    *   *平台*: Wayback Machine (`web.archive.org`)。
*   **元数据提取:**
    *   **ExifTool**: 强大的文件元数据读取、写入和编辑工具。

#### 1.5 网络空间搜索引擎 (Network Space Search Engines)

*   **FOFA, Shodan, ZoomEye, Censys, Quake**: 用于搜索公网资产、服务指纹、漏洞特征、关联 IP/域名等。

### 二、指纹识别与扫描 (Fingerprinting & Scanning)

#### 2.1 技术栈识别 (Technology Fingerprinting)

*   **Wappalyzer**: 浏览器插件和库，识别网站技术栈。
*   **WhatWeb**: 命令行 Web 指纹识别工具。
*   **御剑 WEB 指纹识别**: 图形化指纹识别工具。
*   **CMSmap**: 针对特定 CMS (WordPress, Joomla, Drupal) 的指纹识别和漏洞扫描。
*   **CMSeeK**: CMS 检测和利用套件。
*   **ACMSDiscovery**: CMS检测。
*   **AngelSword**: CMS漏洞检测框架。

#### 2.2 WAF 识别 (WAF Detection)

*   **wafw00f**: 命令行 WAF 指纹识别工具。

#### 2.3 端口扫描 (Port Scanning)

*   **Nmap**: 综合网络扫描和主机发现工具 (端口/服务/版本/OS/脚本)。
*   **Masscan**: 极快的 TCP 端口扫描器 (适合大规模探测)。
*   **Zmap**: 快速无状态网络扫描器。
*   **御剑高速端口扫描工具**: 图形化快速端口扫描。
*   **(可能包含)**: Goby (集成扫描), Yakit (集成扫描)。

### 三、漏洞扫描与利用 (Vulnerability Scanning & Exploitation)

#### 3.1 综合漏洞扫描 (Comprehensive Vulnerability Scanning)

*   **AWVS (Acunetix)**: 商业 Web 应用漏洞扫描器。
*   **Nessus**: 商业网络和系统漏洞扫描器。
*   **AppScan (HCL)**: 商业 Web 应用和移动应用安全测试工具。
*   **Xray**: 被动和主动结合的 Web 漏洞扫描器。
*   **Goby**: 集成资产发现、指纹识别、漏洞扫描利用的工具。
*   **Yakit**: 集成多种功能的交互式安全测试平台。
*   **ZAP (OWASP Zed Attack Proxy)**: 开源 Web 应用安全扫描器和代理。
*   **(针对性)**: 针对特定漏洞的扫描脚本 (如 Nmap NSE 脚本, PoC 脚本)。

#### 3.2 漏洞利用框架与数据库 (Exploitation Frameworks & Databases)

*   **Metasploit Framework**: 流行的开源渗透测试框架。
*   **Searchsploit**: Exploit-DB 数据库的命令行搜索工具。
*   **Cobalt Strike (CS)**: 商业 C2 与后渗透框架 (也用于漏洞利用阶段)。

#### 3.3 Web 漏洞利用 (Web Exploitation)

*   **SQL 注入:**
    *   **SQLMap**: 自动化 SQL 注入检测和利用工具。
    *   **Havij**: (较老) 图形化 SQL 注入工具。
    *   **啊D注入工具**: (较老) 国产图形化 SQL 注入工具。
    *   **超级SQL注入工具【SSQLInjection】**: 国产图形化 SQL 注入工具。
*   **XSS (跨站脚本):**
    *   **BlueLotus_XSSReceiver**: XSS Payload 接收平台/工具。
*   **反序列化:**
    *   **Java**: **ysoserial** (生成 Java 反序列化 Payload)。
    *   **Shiro**: **shiro_attack** (针对 Apache Shiro 框架的检测和利用工具)。
*   **PHP 特定利用:**
    *   **(示例) Bypass-PHP-GD-Process-To-RCE**: 利用特定 PHP 库漏洞的脚本/方法。
* **PYTHON SSTI**
	*   **(实例) 开启图形化界面**: python -m fenjing webui

#### 3.4 数据库利用 (Database Exploitation)

*   **(通用名称) SQLTools**: 可能指数据库客户端或针对性利用工具。
*   **MySQL 综合利用工具**: 图形化工具，集成 UDF/MOF 提权等。
*   **OracleShell**: 针对 Oracle 数据库的利用工具。

#### 3.5 特定 CVE 利用

*   **(示例)**: CVE-2019-1405, CVE-2019-1322, CVE-2022-24521 (Payload_Aabyss) - 这些通常是针对特定漏洞的 PoC 或 Exploit 脚本。

### 四、密码破解与字典 (Password Cracking & Dictionaries)

#### 4.1 在线爆破 (Online Brute Force)

*   **Hydra**: 支持多种协议 (SSH, FTP, RDP, SMB, HTTP Basic/Form 等) 的在线密码爆破。
*   **Medusa**: 类似 Hydra 的在线密码爆破工具。
*   **webcracker**: 可能是针对Web后台的爆破。

#### 4.2 离线哈希破解 (Offline Hash Cracking)

*   **John the Ripper (John)**: 跨平台密码哈希破解工具。
*   **Hashcat**: 基于 GPU 的高速密码哈希破解工具。
*   **hash-identifier**: 识别哈希类型。

#### 4.3 压缩包密码破解 (Archive Password Cracking)

*   **ARCHPR (Advanced Archive Password Recovery)**: 支持多种压缩格式 (ZIP, RAR, 7z 等)。
*   **bkcrack**: 针对传统 ZipCrypto 的已知明文攻击。
*   **Ziperello**: ZIP 密码破解工具。
*   **AdvancedRARPassword**: RAR 密码破解工具。
*   **ZipCenOp**: 修复或分析 ZIP 文件结构以辅助破解 (如伪加密)。

#### 4.4 特定应用/协议密码破解

*   **Oracle**: **orcalpass** (针对 Oracle 密码哈希的破解)。
*   **RDP/VNC 等**: 可使用 Hydra, Medusa 等在线爆破工具。
*   **WiFi**: **Aircrack-ng Suite**, **EWSA (Elcomsoft Wireless Security Auditor)**。
*   **Office**: **Elcomsoft** 系列工具, **Hashcat** (支持 Office 哈希)。

#### 4.5 密码字典与生成器 (Wordlists & Generators)

*   **字典文件示例**: `400W常用密码(整理).txt`, `6000常用密码字典.txt`, `弱口令字典.rar`, `常见密码.txt`
*   **字典生成器**: **简单字典生成器**, **N.C.P.H社会工程学字典生成器**, **真空密码字典生成器**, **亦思社会工程学字典生成器**, **品轩字典生成器**, **易优软件-超级字典生成器**, **黑刀超级字典生成器**。
*   **规则/掩码**: Hashcat 和 John 支持基于规则和掩码的生成。

### 六、内网穿透与代理 (Internal Tunneling & Proxying)

#### 6.1 端口转发 (Port Forwarding)

*   **LCX**: 经典端口转发工具 (如 `htran`)。
*   **Netcat (nc)**: 可通过管道或参数实现简单端口转发。
*   **SSH Tunneling (-L, -R, -D)**: 利用 SSH 实现本地/远程端口转发和 SOCKS 代理。

#### 6.2 HTTP/S 隧道 (HTTP/S Tunneling)

*   **reGeorg / neoreg**: 将 SOCKS 代理流量封装在 HTTP/S 请求中。
*   **reDuh**: 利用 HTTP/S 进行端口转发和连接管理。

#### 6.3 多功能隧道/SOCKS 代理 (Multi-functional Tunneling / SOCKS Proxy)

*   **EarthWorm (ew)**: 支持正/反向 SOCKS 代理、端口转发、多级级联。
*   **FRP**: 高性能反向代理，支持 TCP, UDP, HTTP, HTTPS。
*   **NPS**: 类似 FRP 的内网穿透代理服务器。
*   **Venom**: 支持多级代理和端口转发。
*   **SSH Tunneling (-D)**: 创建 SOCKS 代理。

#### 6.4 DNS 隧道 (DNS Tunneling)

*   **Iodine**: 通过 DNS 查询进行隧道传输。

#### 6.5 其他代理/穿透工具

*   **Powercat**: PowerShell 版 Netcat，支持反向/正向 shell, 文件传输, 端口转发。

### 七、命令与控制 / 后渗透 (C2 / Post-Exploitation)

*   **(简单 C2)**: Netcat, Powercat, PoshC2, Sliver 等。

    *   **dnSpy**: .NET 程序集编辑器、反编译器和调试器。
    *   **ILSpy**: 开源 .NET 反编译器。

    *   **Easy Python Decompiler**: (可能是) PyInstaller/PyArmor 等打包程序的解包/反编译工具。
    *   **pyinstxtractor**: 解包 PyInstaller 生成的 exe。

#### 9.2 调试器 (Debuggers)

*   **Windows**:
    *   **OllyDbg (OD)**: 经典的 32 位用户态调试器。

    *   **GDB**: GNU Debugger。

*   **跨平台/其他**: **IDA Pro**, **Ghidra** 也包含调试功能。

#### 9.3 PE/ELF 分析工具 (PE/ELF Analysis Tools)

#### 11.2 数字取证与数据恢复 (Digital Forensics & Data Recovery)

*   **文件恢复/雕刻 (File Carving):**
    *   **Foremost**: 基于文件头/尾特征的文件恢复工具。
    *   **Scalpel**: Foremost 的一个改进版本。
    *   **Bulk Extractor**: 扫描磁盘镜像提取特定信息（邮件、URL 等）。
*   **磁盘镜像/分析:**
    *   **FTK Imager**: 创建磁盘镜像，浏览文件系统。
    *   **Autopsy**: 开源数字取证平台 (基于 Sleuth Kit)。
    *   **Volatility Framework**: 内存镜像分析框架。
    *   **Elcomsoft Forensic Disk Decryptor**: 解密加密磁盘/卷。
*   **日志分析:**
    *   **Log Parser Lizard**: GUI for Microsoft Log Parser 2.2.
    *   **Web日志安全分析工具**, **秋式网站日志分析器**: 针对 Web 日志的分析工具。
